Introdução

Esta documentação o ajudará a se familiarizar com os recursos da Rick e Morty API e lhe mostrará como fazer a consulta dos dados, para que você possa obter o máximo dela.

REST

Url base: http://localhost:8080/api

A Url base contém informações sobre todos os recursos API disponíveis, onde todas as requisições possíveis são GET. Todas as respostas das requisições GET retornarão dados em JSON.

GET

http://localhost:8080/api

{

"personagens"

:

"http://localhost:8080/personagem"

,

"localizacao"

:

"http://localhost:8080/localizacao"

,

"episodios"

:

"http://localhost:8080/episodios"

}

Esses são os três recursos disponíveis:

  • Personagem: Use para listar todos os personagens.
  • Localização: Use para listar todas as localizações.
  • Episódio: Use para listar todos os episódios.

    • Personagem

    Há um total de 30 personagens ordenados por id.

    Tabela de personagem

    Campo Tipo Descrição
    id int O id do personagem
    name String O nome do personagem
    status Status O estado do personagem (Vivo, Morto ou Desconhecido)
    species String A espécie do personagem
    type String O tipo ou sub-espécie do personagem
    gender Gender O gênero do personagem (Feminino, Masculino, Sem gênero ou Desconhecido)

    Enumerador de tipo

    Campo Descrição
    1 Vivo
    2 Morto
    3 Desconhecido

    Enumerador de gênero

    Campo Descrição
    1 Feminino
    2 Masculino
    3 Sem gênero
    4 Desconhecido

    Filtrar personagens

    Você também pode buscar por personagens utilizando outros filtros, como:

    name: filtra pelo nome fornecido: /personagem/name/Rick Sanchez

    status: filtra pelo estado fornecido (Vivo, Morto ou Desconhecido): /personagem/status/Vivo
